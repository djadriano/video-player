# Video Player

A video player that play the videos in sequence from the library created by user.

# Demo

[https://h2q3f.sse.codesandbox.io/](https://h2q3f.sse.codesandbox.io/)

## Stack

- Node.js
- React 16.8.6
- Sass

## How to use

Install the dependencies:

```bash
$ yarn install
```

Running the project:

```bash
$ yarn dev
```

Build the project:

```bash
$ yarn build
```
